常见问题

负责人通常会问这些

阿凌是怎么帮团队省人力的？

把「开 PDF → 对照 JD → 记分 → 填表 → 回 Boss」压成「上传 → 自动排序 → 守护跟进」。同样一个人，能跟进更多高匹配候选人，而不是先加人。

数据安全怎么样？会不会上传到外部平台？

默认本地部署。简历解析与评分在你们环境内完成，适合对简历敏感、不愿上云的团队。具体网络与 API 配置可在演示时说明。

会不会把所有人都刷成高分？

不会。按岗位硬性 / 软性要求与权重打分，详情里能看技能亮点与未达标项；管理侧也可导出结果复盘，而不是黑箱。

Boss 自动回复会不会像机器人？

可配置正式 / 轻松 / 自定义语气；支持每日上限与间隔，避免失控群发。演示时可按你们话术试一轮。

第一次怎么验证值不值得上？

建议先远程演示：拿最近真实投递的一批简历跑通排序与导出。效果对得上你们节奏，再谈部署与日常使用。

怎么衡量有没有用？

看三件事：初筛是否还加班、高匹配是否更快进到面、Boss 私信是否还占满白天。报表可导出，便于周会对齐。
